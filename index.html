About

tweeter

tweeter is a data engineering project I worked on that uses the Twitter API, Python, and Airflow to extract tweets from the API and land the data in a local Postgres table. Upon landing the data, I then performed some basic analysis and visualization to look at what words are most tweeted.

coviz

coviz was a Shiny application I developed using R to visualize COVID-19 data in real-time. Using the data uploaded to The New York Times Github repository, I dynamically read and processed the data and then created a visual interface to track trends and statistics around the spread of the virus.

Track Power Rankings

An R Shiny application that creates auto generated power rankings for indoor NCAA track & field (distance events only). The application performs web-scraping of data and is able to extract, clean and transform the data into a usable state for generating "rank scores".
